Key Findings
Answering the Research Questions
📈
As gamification goes up, does violence go up?
Yes — moderately. r = 0.327, p < 10⁻²⁰⁰. Largest jump is 0→1 (+1.64 pts). Second surge at 6–9 driven by drone-strike content.
🔍
How much does gamification explain?
Only 10.7% of variance. Dehumanization (r=0.57) is a far stronger predictor. Gamification acts partly through dehumanization (r = 0.58).
🎯
What image type scores high on both?
Drone strike content only — mean gam = 6.88, violence = 6.81. No other type exceeds gam = 2.5 while also scoring high on violence.
⚠️
Is the relationship linear?
No — threshold effect. Gam 1–5 plateau ~4.7–5.1. Only at gam ≥ 6 does violence climb sharply, suggesting a non-linear dynamic.
📺
Does the channel matter?
Yes. Wargonzo, rusich_army, ZA_FROHT average gam ≥ 2.0. Hyevuy_dnepr (0.68) stays low on gamification yet still hits moderate violence.
🧮
What does the equation say?
ŷ = 3.67 + 0.37x — even at gam=0, baseline violence is 3.67 (war context floor). Each +1 gam point adds 0.37 violence points.

Cell 2 — Scatter + Regression Line
import matplotlib.pyplot as plt import numpy as np from scipy import stats df_c = df[['gamification_score', 'implied_violence_score']].dropna() x, y = df_c['gamification_score'].values, df_c['implied_violence_score'].values slope, intercept, r, p, se = stats.linregress(x, y) sample = df_c.sample(3000, random_state=42) xj = sample['gamification_score'] + np.random.uniform(-0.3,0.3,len(sample)) yj = sample['implied_violence_score'] + np.random.uniform(-0.3,0.3,len(sample)) fig, ax = plt.subplots(figsize=(10,6), facecolor='#0a0e1a') ax.set_facecolor('#0f1628') ax.scatter(xj, yj, alpha=0.12, s=12, color='#005BBB') xl = np.linspace(0,10,100) ax.plot(xl, intercept+slope*xl, color='#FFD500', lw=2, label=f'ŷ={intercept:.2f} + {slope:.3f}x R²={r**2:.3f}') ax.tick_params(colors='#7a90b8') for spine in ax.spines.values(): spine.set_edgecolor('#1a2a4a') ax.set_xlabel('Gamification Score', color='#7a90b8') ax.set_ylabel('Implied Violence Score', color='#7a90b8') ax.set_title('Gamification vs Implied Violence', color='#e8eef8') ax.legend(facecolor='#0f1628', labelcolor='#e8eef8') plt.tight_layout(); plt.show()
Cell 3 — Mean Line + 95% CI Ribbon
means = df_c.groupby('gamification_score')['implied_violence_score'].agg(['mean','std','count']) ci = 1.96 * means['std'] / np.sqrt(means['count']) fig, ax = plt.subplots(figsize=(10,5), facecolor='#0a0e1a') ax.set_facecolor('#0f1628') ax.plot(means.index, means['mean'], 'o-', color='#005BBB', lw=2, ms=7) ax.fill_between(means.index, means['mean']-ci, means['mean']+ci, alpha=0.2, color='#FFD500') ax.tick_params(colors='#7a90b8') for spine in ax.spines.values(): spine.set_edgecolor('#1a2a4a') ax.set_xlabel('Gamification Score', color='#7a90b8') ax.set_ylabel('Mean Implied Violence', color='#7a90b8') ax.set_title('Mean Implied Violence by Gamification Level (95% CI)', color='#e8eef8') ax.set_xticks(range(11)); plt.tight_layout(); plt.show()
Cell 4 — Content Type Bubble Chart
ct = df.groupby('content_type')[['gamification_score','implied_violence_score']].mean() ct['count'] = df.groupby('content_type').size() fig, ax = plt.subplots(figsize=(10,7), facecolor='#0a0e1a') ax.set_facecolor('#0f1628') ax.scatter(ct['gamification_score'], ct['implied_violence_score'], s=ct['count']/ct['count'].max()*2000, color='#005BBB', edgecolors='#FFD500', lw=1.2, alpha=0.8) for idx, row in ct.iterrows(): ax.annotate(idx,(row['gamification_score'],row['implied_violence_score']), fontsize=9,ha='center',va='bottom',color='#e8eef8') ax.tick_params(colors='#7a90b8') for spine in ax.spines.values(): spine.set_edgecolor('#1a2a4a') ax.set_xlabel('Mean Gamification Score', color='#7a90b8') ax.set_ylabel('Mean Implied Violence Score', color='#7a90b8') ax.set_title('Content Types — Gamification vs Violence (bubble = count)', color='#e8eef8') plt.tight_layout(); plt.show()
